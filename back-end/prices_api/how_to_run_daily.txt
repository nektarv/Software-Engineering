ΟΔΗΓΙΕΣ: ΑΥΤΟΜΑΤΗ ΕΚΤΕΛΕΣΗ SCRIPT (TASK SCHEDULER)

Ακολουθήστε αυτά τα βήματα για να ρυθμίσετε το script ώστε να τρέχει αυτόματα κάθε μέρα και να κατεβάζει τις τιμές ρεύματος.

-----------------------------------------------------
ΒΗΜΑ 1: ΒΡΕΙΤΕ ΤΗ ΔΙΑΔΡΟΜΗ ΤΗΣ PYTHON
-----------------------------------------------------
Πριν ξεκινήσετε, πρέπει να μάθετε πού ακριβώς είναι εγκατεστημένη η Python στον υπολογιστή σας.

1. Πατήστε το κουμπί Start και γράψτε "cmd" για να ανοίξετε τη γραμμή εντολών.
2. Γράψτε την εντολή:
   where python
3. Θα σας εμφανίσει μια διαδρομή (π.χ. C:\Users\Name\AppData\Local\Programs\Python\Python39\python.exe).
4. Αντιγράψτε αυτή τη διαδρομή. Θα τη χρειαστείτε παρακάτω.

-----------------------------------------------------
ΒΗΜΑ 2: ΡΥΘΜΙΣΗ ΤΟΥ TASK SCHEDULER
-----------------------------------------------------
1. Πατήστε Start, γράψτε "Task Scheduler" (ή "Χρονοδιάγραμμα Εργασιών") και ανοίξτε το.
2. Στο δεξί μενού, πατήστε "Create Basic Task" (Δημιουργία Βασικής Εργασίας).
3. Name: Δώστε ένα όνομα, π.χ. "Update Electricity Prices" και πατήστε Next.
4. Trigger: Επιλέξτε "Daily" (Καθημερινά) -> Next.
5. Time: Ορίστε ώρα 17:00:00 (5:00 MM) -> Next.
   (Σημείωση: Οι τιμές της επόμενης μέρας ανακοινώνονται συνήθως μέχρι τις 14:30, οπότε στις 17:00 είμαστε σίγουροι).
6. Action: Επιλέξτε "Start a program" -> Next.

-----------------------------------------------------
ΒΗΜΑ 3: ΠΡΟΣΟΧΗ ΣΤΑ PATHS (ΣΗΜΑΝΤΙΚΟ)
-----------------------------------------------------
Εδώ πρέπει να συμπληρώσετε τα πεδία προσεκτικά για να δουλέψει σωστά.

1. ΠΕΔΙΟ "Program/script":
   Κάντε ΕΠΙΚΟΛΛΗΣΗ τη διαδρομή της Python που βρήκατε στο Βήμα 1.
   Παράδειγμα: C:\Users\User\AppData\Local\Programs\Python\Python313\python.exe

2. ΠΕΔΙΟ "Add arguments (optional)":
   Εδώ βάζετε την ΠΛΗΡΗ ΔΙΑΔΡΟΜΗ του αρχείου .py, ΜΕΣΑ ΣΕ ΕΙΣΑΓΩΓΙΚΑ.
   Μορφή: "C:\Path\To\Your\Project\day_ahead_prices.py"
   Παράδειγμα: "C:\Users\User\Documents\Project\prices_api\day_ahead_prices.py"

3. ΠΕΔΙΟ "Start in (optional)":
   Εδώ βάζετε τον ΦΑΚΕΛΟ που περιέχει το script (χωρίς το όνομα του αρχείου και ΧΩΡΙΣ εισαγωγικά).
   Σκοπός: Αυτό εξασφαλίζει ότι το αρχείο prices.csv θα αποθηκευτεί δίπλα στο script.
   Παράδειγμα: C:\Users\User\Documents\Project\prices_api

-> Πατήστε Next και μετά Finish.

-----------------------------------------------------
ΒΗΜΑ 4: ΔΟΚΙΜΗ
-----------------------------------------------------
Για να βεβαιωθείτε ότι όλα λειτουργούν:

1. Στο αριστερό μενού του Task Scheduler, πατήστε "Task Scheduler Library".
2. Βρείτε την εργασία που φτιάξατε στη λίστα.
3. Κάντε Δεξί Κλικ πάνω της και επιλέξτε "Run".
4. Πηγαίνετε στον φάκελο του script και ελέγξτε αν η ημερομηνία τροποποίησης του αρχείου prices.csv άλλαξε μόλις τώρα.

Αν ενημερώθηκε, το σύστημα είναι έτοιμο.



ΟΔΗΓΙΕΣ ΓΙΑ ΤΟ 2ο TASK (UPDATE DATABASE)

1. Ανοίξτε το "Task Scheduler".
2. Πατήστε "Create Basic Task" (Δημιουργία Βασικής Εργασίας).
3. Name: Δώστε ένα όνομα, π.χ. "Update DB with Prices" και πατήστε Next.
4. Trigger: Επιλέξτε "Daily" (Καθημερινά) -> Next.
5. Time: Ορίστε ώρα 17:15:00 -> Next.
   (Σημαντικό: Να είναι ΜΕΤΑ τις 17:00 που τρέχει το πρώτο script).
6. Action: Επιλέξτε "Start a program" -> Next.
